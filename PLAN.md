## Site Structure

### Pages
1. **Homepage** — Live odds dashboard, featured matches, trending analysis
2. **Groups (A-L)** — Each group page with team profiles, odds, predictions
3. **Match Center** — Individual match pages with odds comparison + AI analysis
4. **Odds Comparison** — Side-by-side bookmaker odds for every match
5. **Arb Alerts** — Flagged arbitrage opportunities
6. **Predictions** — AI-generated probability models
7. **Team Profiles** — All 48 teams with squad, form, history
8. **Blog/News** — SEO content, betting guides, World Cup news
9. **Group Simulator** — Interactive "what if" tool
10. **About** — What is KickScan, how AI works

### Tech Stack
- **Frontend:** Next.js (React) — fast, SEO-friendly, free hosting on Vercel
- **Backend:** Node.js API routes
- **Database:** Supabase (free tier) for odds data, match info
- **Odds Data:** Scrape from public APIs (odds-api.com has free tier)
- **Hosting:** Vercel (free)
- **Domain:** kickscan.io (purchased)
