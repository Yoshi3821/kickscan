## 🛠️ CAPCUT AUTOMATION WORKFLOW

### STEP 1: PROJECT SETUP (30 seconds)
1. **Open CapCut** → **New Project**
2. **Aspect Ratio:** 9:16 (TikTok vertical)
3. **Duration:** 16 seconds target
4. **Import** all Runway videos to media library

### STEP 2: BULK VIDEO IMPORT (1 minute)
```
□ Drag Video 1 to timeline (0-4s)
□ Drag Video 2 to timeline (4-8s)  
□ Drag Video 3 to timeline (8-12s)
□ Drag Video 4 to timeline (12-16s)
□ Add transitions between clips (0.3s fade)
```

## 🚀 AUTOMATION SHORTCUTS

### CAPCUT HOTKEYS (Speed up editing)
- **Space:** Play/Pause
- **Ctrl+Z:** Undo
- **Ctrl+C/V:** Copy/Paste elements
- **Delete:** Remove selected
- **Ctrl+T:** Add text
- **Ctrl+M:** Add music

### TEMPLATE SAVE/LOAD
1. **Create perfect template** with all settings
2. **Save as "KickScan Template 1"**
3. **Export template** for reuse
4. **Load template** for new videos  
5. **Just replace videos + update text**

### BATCH EXPORT SETTINGS
- **Resolution:** 1080p (TikTok optimal)
- **Frame Rate:** 30fps (smooth playback)
- **Quality:** High (no compression loss)
- **Format:** MP4 (universal compatibility)
- **Filename:** "KickScan_YYYYMMDD_[topic]"
